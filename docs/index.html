ResourceKit

A Swift-native iOS framework for working with files and file types.

Installation

I plan to offer in the near future the option to download ResourceKit via CocoaPods or Carthage. Until then, copy the ResourceKit folder to your Xcode project. When you do, enable the following options:

  1. Copy items if needed
  2. Create groups
